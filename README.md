# Installation
run `npm i`

# Running
run `npm run dev`

## API Documentation

The API specifications are available via Swagger UI.

1.  **Start the server:** Run `npm run dev` in your terminal.
2.  **Access Swagger UI:** Open your browser and navigate to `http://localhost:5000/api-docs` (assuming the server is running on port 5000).

## Deploying

Do either:
1. Inside Production server run `git clone https://github.com/2110503-CEDT68/be-project-68-sleepnpray`
2. setup automated action to run `npm i` and then `npm start`

or
1. In your local repository run `git archive --format=zip --output=app.zip HEAD`
2. drop the resulting zip file to production server
